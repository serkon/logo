Menu Service

Uygulamanıza ait menüler Logo PaaS platformunda hizmet veren diğer uygulamaların menüleri ile birlikte aynı ortamda kullanıcıya sunulsun. Kullanıcılar yetkisi dahilinde olan farklı uygulama menüleri ile dilediği şekilde masaüstü oluştursun.

KEŞFET

Kullanım Senaryoları

Logo PaaS platformunda ki uygulamalar için kullanıcı dostu bir giriş kapısıdır.

  • Uygulamalarınızın Menülerini API yoluyla ya da Arayüzlerden hızlıca kayıt edin
  • Kullanıcılarınız menü tanımlarınız üzerindeki yetkiler doğrultusunda görebilsin
  • Uygulamanıza ait Yönetici ekranlarınızı Logo PaaS yönetici yetkilendirme yapısı ile ve API yoluyla hızlıca kayıt edin
  • Javascript mesajlaşması ile uygulamanız menü servis arayüzü ile etkileşim içinde olsun
  • Çalışma esnasında kişiye özel menüleriniz varsa sağlayacağınız bir API ucuyla menü servise bu menüleride aktarın
  • Yetkilendirme kontrolünü dilerseniz menü servis sizin uygulamanızdan kullansın
  • Kişisel masaüstünüzü oluşturup, sık kullandığınız menülere hızlıca ulaşın
