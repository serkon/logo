Dashboard Feature

Farklı veri kaynaklarından (rest API, Veritabanı) çekilen verilerinizi görsel bileşenlere dönüştürün, Çoklu hesap (Multitenancy) desteğinden faydalanarak tek bir dashboard tanımı yapın, tasarladığınız dashbordu herkes kendi verisi ile görsün.

KEŞFET

Kullanım Senaryoları

Dashboard Servis Özelliğini Nasıl Kullanacağınızı Öğrenin!

  • Farklı tiplerde ( Rest API – Veritabanı) veri kaynakları tanımlayın
  • Veri kaynağı tanımlarınızı parametrik olarak (kullanıcıya göre, hesaba göre) yapın
  • Parametrik filtreler tanımlayın
  • Kendi uygulamanız içerisinde tasarladığınız dashboardu gömülü olarak açın
