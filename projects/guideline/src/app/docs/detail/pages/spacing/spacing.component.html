Spacing

Spacing is one of the most important principles of design. In order to create accurate and consistent visual hierarchies, 2 different spacing scales were defined.

Scales

We have 2 different spacing scales. One is used to determine the hierarchy between the elements of a component and the other is used to define spacing between page elements.

Spacing Purpose
Spacing Scale Used in smaller spaces, more detailed spacing needs, especially within a component. (i.e. the space between text inbut and component title)
Layout Scale It is mainly used when positioning components within the page. (i.e. the space between header and a form component)
