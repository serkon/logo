Overview

Button groups are often used to switch between different views of the same content. It can also be used from occasionally to narrow down the same data in different ways.

When to use

Button groups are often used to switch between different views of the same content. It can also be used from occasionally to narrow down the same data in different ways.

When not to use

Tabs

If you can not create a breakdown within the data you want to show to the user or if you want to position different types of content that are not similar to each other in a single field, use tabs instead of button groups.

Switch

In cases where you want the user to make a simple "yes" "no" choice, it is better to use the switch component.
