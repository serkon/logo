Tag

Tags are small indicators and so well-suited for cases where you want to add information but don’t want it grab as much attention as a button.

Overview

Tags allow users define attributes that can be added or removed to a content. Tags are also used to select multiple values from a large set of predefined options.

When to use

- Use them to show and edit labels and categories.

- Use tags when content is mapped to multiple categories and user needs to edit them.

When not to use

- Don’t use for decorative purpose.

- Don’t use for navigation.

Formatting

Anatomy

Behavior

The user can click and search for the option that he/she are looking for and select the option from the popover list.

Button

Tags can be added and removed and change state, but they don’t have actions that affect other things on their own. To offer an action in a simple component, consider a button.
