      Wizard has two types: vertical and horizontal wizards.
    </p>

      <li>Overview</li>

  <div class="overview">
    <h2>Overview</h2>
    <p>
      The wizard aims to help users by dividing large or complex tasks into segments. Use the wizard if the user has to
      accomplish a long task (such as filling out a long questionnaire) or a task that is unfamiliar to the user. The
      flow should consist of a minimum of 3 and a maximum of 8 steps.
    </p>
  </div>
  <div class="wtu">
    <h3>When to use</h3>
    <p>
      -The wizard can be used for both create and edit scenarios. If your application contains both, consider using the
      same means for both scenarios.
    </p>
  </div>
  <div class="wtnu">
    <h3>When not to use</h3>
    <p>
      -If you have a task with only 2 steps or a format that the user is familiar with (for example, it is part of their
      daily routine), do not use the wizard as it only adds unnecessary clicks to the process.
    </p>
  </div>
  <div class="types">
    <table>
      <tr>
        <th>Type</th>

      </tr>
      <tr>
        <td>Horizontal wizard</td>
        <td>
          Horizontal wizard can be used when the step detail contains a wide layout, such as table.
        </td>
      </tr>
      <tr>
        <td>Vertical wizard</td>
        <td>
          Vertical wizard can be used when the step detail has no wide content.
        </td>
      </tr>
