      <div class="overview">
        <h2>Overview</h2>
        <p>Actions can be used as follows.</p>
        <br/>
        <p>
          -They can be independent of the current selection and not related to a specific item or object.
        </p>
        <p>
          -They can be specific to the current object (user selects one item).
        </p>
        <p>
          -They can apply to a set of items (user selects two or more items).
        </p>
        <p>
          -They can control the settings for parts of the UI content. For example, an action can affect all items in a
          table.
        </p>
        <p>
          -The action bar is mostly used for buttons (with an icon or text). The buttons are right-aligned by default. But
          depending on the action bar, buttons may be center aligned or left aligned.
        </p>
      </div>
