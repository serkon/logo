    <div class="overview">
      <h2>Overview</h2>
      Radio buttons are often arranged in a group of at least two options. A user may choose only a single option or
      radio
      button among the set or group.
    </div>
    <div class="wtu">
      <h3>When to use</h3>
      <p>
        - When there is a list of two or more options that are mutually exclusive and the user must select exactly one
        choice.
      </p>
      <p>
        - Use when you want to see all available options.
      </p>
    </div>
    <div class="wtnu">
      <h3>When not to use</h3>
      <p>- Radio buttons should be used instead of checkboxes if only one item can be selected from a list.</p>
    </div>
