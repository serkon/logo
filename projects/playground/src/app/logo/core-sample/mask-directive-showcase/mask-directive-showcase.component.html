RegExp without condition

With this regular expression, the following input accepts two alpha characters at first, then it accepts up to 10 numeric chars. After the second character, it will automatically insert the "-" character in the input field.

[regexp]="'^[a-zA-Z]"{{'{'}}"1,2}$|^[a-zA-Z]\"{{'{'}}"1,2}[0-9]*$'"

RegExp with condition

Same sample but it also accept number at second position

[conditions]="'^[a-zA-Z]"{{'{'}}"1,2}$|^[a-zA-Z]\"{{'{'}}"1,2}[0-9]*$'"
